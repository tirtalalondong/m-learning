Basic 1 - Tentang Framework

Website

Website atau situs dapat diartikan sebagai kumpulan halaman-halaman yang digunakan untuk menampilkan informasi teks, gambar diam atau gerak, animasi, suara, dan atau gabungan dari semuanya itu baik yang bersifat statis maupun dinamis yang membentuk satu rangkaian bangunan yang saling terkait dimana masing-masing dihubungkan dengan jaringan-jaringan halaman atau hyperlink.

Framework

Framework adalah kerangka-kerja yang meruapakan sekumpulan fungsi, class dan aturan-aturan. Berbeda dengan library yang sifatnya untuk tujuan tertentu saja, framework bersifat menyeluruh mengatur bagaimana kita membangun aplikasi.

MVC

Models-Views-Controllers, merupakan motede yang memisahkan pengembangan aplikasi berdasarkan komponen utamanya. Seperti manipulasi data (Model), antarmuka pengguna (Views) dan bagian yang menjadi kontrol (Controllers) dalam sebuah aplikasi.

Python

Python dapat diartikan sebagai bahasa pemrograman tingkat tinggi yang interpreted dan berorientasi objek. Merupakan bahasa pemrograman yang dibangun dengan struktur data tingkat tinggi, menggabungkan antara dynamic typing dan dynamic binding yang membuatnya menjadi semakin baik untuk pengembangan aplikasi secara cepat.

Web2py

Web2py adalah framework aplikasi web berbasis open source yang digunakan untuk pengembangan secara cepat, berskala, aman dan untuk aplikasi database-driven berbasis web yang portable. Framework web2py di tulis serta di jalankan menggunakan bahasa pemrograman Python dan dapat dijalankan secara langsung.
